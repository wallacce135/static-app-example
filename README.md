# Static (HTML / CSS / JS)

Пример статического приложения (HTML / CSS / JS), которое можно развернуть в **Timeweb Cloud Apps** без настройки.

:rocket: [Создать свой Apps](https://timeweb.cloud/my/apps/create)

:books: [Документация Timeweb Cloud Apps](https://timeweb.cloud/docs/apps)

## <a name="dev"></a>Локальный запуск проекта

**Проект не имеет возможности сборки.**

Просмотр содержимого осуществляется через открытие index.html любым браузером
